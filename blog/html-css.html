What are the best practices associated with using classes vs ids?

Classes and ids are used to define or change the layout, appearance and design of specific elements coded in HTML. A class is best used to define style where it may be applied to multiple elements on the page. An ID is appropriate where unique styling is needed for an element, meaning more than one element with the same ID cannot be used in an HTML document.
